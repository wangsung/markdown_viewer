# 📝 마크다운 뷰어 & 에디터 (Markdown Viewer & Editor)

> **독립적(Offline-ready)으로 동작하는 고성능 스크롤 동기화 지원 현대적 마크다운 편집기**

본 프로젝트는 외부 웹 API나 네트워크 요청 없이 완벽히 로컬 웹 리소스만으로 작동하며, 웹 브라우저 상에서 미려하게 마크다운 문서를 작성하고 즉시 렌더링 결과를 볼 수 있는 독립형 반응형 뷰어입니다.

---

## 🚀 시작하기 (How to Run)

본 프로젝트는 순수 클라이언트용 웹 어플리케이션으로 별도의 서버 빌드 및 `npm install` 설치가 필요하지 않습니다.

1. 본 저장소의 모든 소스코드를 다운로드하거나 클론합니다.
2. `markdown_viewer.html` 파일을 웹 브라우저(Chrome, Edge 등)로 **더블 클릭하여 실행**합니다.

---

## ✨ 주요 기능 (Key Features)

### 1. ⚙️ 에디터-프리뷰 쌍방향 스크롤 동기화
* 에디터 창과 프리뷰 창의 스크롤 위치가 쌍방향으로 정밀하게 동기화됩니다. 에디터를 편집하거나 스크롤할 때 우측 프리뷰 화면이 부드럽고 정확하게 따라 움직여 흐름을 잃지 않고 문서를 열람하고 작성할 수 있습니다.
* 마우스 드래그를 이용해 에디터 텍스트를 광범위하게 선택할 시, 프리뷰 화면이 커서가 가리키는 라인에 자동으로 최적 정렬되는 지능형 정밀 동기화 알고리즘이 적용되어 있습니다.

### 2. 💾 간단한 MD 편집과 저장
* 에디터 상단에 위치한 서식 도구 모음(Toolbar)을 사용하여 클릭 한 번으로 빠르게 굵게, 기울임, 헤더, 표, 링크 등을 삽입할 수 있습니다.
* 파일명 표시 뱃지 우측의 **저장** 버튼을 누르면 작성 중인 파일명이 담긴 문서가 로컬 PC 다운로드 폴더로 즉시 저장됩니다.

### 3. 🎨 에디터 내 Heading 스타일링 적용
* 에디터 자체적으로 마크다운 Heading 문법(# 헤더, ## 헤더 등)을 감지하여 시각적으로 개별 스타일링(글자 크기, 두께, 색상)을 적용함으로써 작성 중인 문서의 구조를 쉽게 한눈에 알아볼 수 있습니다.
* 어두운 배경(Slate 900)에 최적화하여 가독성을 높였으며, 리스트/불릿 항목 또한 눈에 잘 띄는 밝은 그레이 색상으로 조정되어 있습니다.

### 4. 📐 수학 공식 및 다이어그램 지원 (LaTeX & Mermaid)
* **수식(KaTeX)**: `$수식$`(인라인) 및 `$$수식$$`(블록) 마크다운 문법을 완벽히 인식하여 아름다운 LaTeX 수학 공식으로 실시간 렌더링합니다.
* **다이어그램(Mermaid)**: `mermaid` 코드 블록을 인식하여 플로우차트, 시퀀스 다이어그램 등의 다이어그램을 브라우저 상에 SVG 그래픽으로 비동기 렌더링합니다.
* 상단 메뉴바의 토글 스위치로 수식 및 다이어그램 지원 기능을 손쉽게 ON/OFF 할 수 있습니다.

### 5. 🔌 완전한 독립형 로컬 동작 (Offline-ready)
* 마크다운 파서(`marked.min.js`), 구문 강조 라이브러리(`highlight.min.js`), 수식 엔진(`KaTeX`), 다이어그램 엔진(`Mermaid`), 에디터 엔진 등을 모두 로컬 `libs/` 디렉터리에 내장하여 외부 인터넷 연결이 전혀 없는 폐쇄망 환경에서도 100% 정상 작동합니다.

### 6. 🗂️ 드래그 앤 드롭 파일 로더 (Drag & Drop Load)
* PC에 있는 마크다운 파일(`.md`, `.markdown`, `.txt` 등)을 에디터 창에 던져 넣으면 브라우저 탭 이동 없이 즉시 파일을 읽어옵니다.
* 작성 중이던 수정 이력(`isDirty = true`)이 있을 시 로컬 파일 다운로드(저장) 여부를 먼저 묻는 **안전 확인(Confirm) 대화상자**를 통해 실수로 작성 중인 소중한 데이터를 유실하지 않도록 보호합니다.

## 📂 프로젝트 파일 구조 (File Structure)

```bash
markdown_viewer/
├── libs/
│   ├── codemirror/           # 에디터 코어 라이브러리 (JS, CSS, Modes)
│   ├── katex/                # LaTeX 수식 렌더링 라이브러리 (JS, CSS, Fonts)
│   ├── github.min.css        # 코드 구문 강조 스타일 테마 (GitHub)
│   ├── highlight.min.js      # 코드 구문 강조 파서
│   ├── marked.min.js         # 마크다운 기본 파서 엔진
│   └── mermaid.min.js        # 다이어그램 그래픽 엔진
├── .agents/                  # AI 에이전트 프로젝트 전용 규칙 디렉터리
├── .gitignore                # 깃 관리 제외 설정 파일
├── README.md                 # 프로젝트 설명 문서 (본 파일)
├── app.js                    # 메인 어플리케이션 스크립트 및 동기화 엔진
├── markdown_viewer.html      # 에디터 UI 및 레이아웃 마크업 구조
└── style.css                 # UI 디자인 및 레이아웃 커스텀 CSS 스타일
```
